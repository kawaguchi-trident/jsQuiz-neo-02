# jsQuiz-neo-02

「formの値操作」の振り返りです。

## 課題内容

下記の「申し込みフォーム」のように、buttonをclickすると、form内の値を取得し、確認画面のように表示されてるようにしてください。

![jsQuiz02サンプル](./sample.gif)

1. `students/{自分の番号}/index.html` を編集して解答してください。
2. 解答は、`<script></script>`内だけでOKです。
3. jQuery利用も可です。
4. 考え方の手順も最下部に準備しました。最初は自分で考えてみてください。
5. jQueryを利用する場合は、$(function(){})内に書きましょう。

### 制作手順

最初は、自分で考えてみましょう。※jQueryの場合は、省略できます。

①必要な要素を、変数に代入していきます。
②クリックイベントを設定します。
③新しく`<ul>`を作成し、result内に設置します。
④inputは、3つあるのでできればループで取得し、`<li></li>`内に設置します。
⑤ループ内でその`<li>`を、`<ul>`内にそれぞれいれて表示させます。

これで、実現できるはずです。
